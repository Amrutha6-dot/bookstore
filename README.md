# bookstore
This is a responsive online bookstore web application built using HTML, CSS, JavaScript, and Bootstrap. The project showcases front-end development skills, including dynamic UI, user interactions, and data handling without a backend.
Users can:
 -> Browse books with dynamic listings
 -> Search by title or author
 -> Filter books by category
 -> Add books to the cart
 -> View a checkout page with total price calculation
This project demonstrates the ability to build user-friendly, responsive web applications suitable for high-traffic scenarios.

Tech Stack
Front-End: HTML5, CSS3, JavaScript
Framework: Bootstrap 5
Data Storage: LocalStorage (for cart persistence)
Features: Responsive UI, Dynamic Book Listing, Search & Filter, Checkout Page
